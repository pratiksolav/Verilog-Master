Code Converter - Binary to Gray

Gray code is an ordering of the binary numeral system such that two successive values differ in only one bit (binary digit).

Gray codes are very useful in the normal sequence of binary numbers generated by the hardware that may cause an error or ambiguity during the transition from one number to the next.

So, the Gray code can eliminate this problem easily since only one bit changes its value during any transition between two numbers.

Gray code finds its applications in Error Detection and Karnaugh Maps.

Hence , Code Converters can be useful in finding the Gray Code Equivalent for the given Binary Number.

We will obtain Boolean Expression as :

G3 = B3 ;

G2 = B3 ⊕ B2 ;

G1 = B2 ⊕ B1 ;

G0 = B1 ⊕ B0 ;

Verilog Codes:


Gate Level:

module binary_to_gray_4bit_gate(bin , gray);
    input [3:0]bin;
    output [3:0]gray;
    
    buf B1(gray[3] , bin[3]);
    xor E1(gray[2] , bin[3] , bin[2]);
    xor E2(gray[1] , bin[2] , bin[1]);
    xor E3(gray[0] , bin[1] , bin[0]);
    
endmodule

Dataflow Level:

module binary_to_gray_4bit_dataflow(bin , gray);
    input [3:0]bin;
    output [3:0]gray;
        
    assign gray[3] = bin[3];
    assign gray[2] = bin[3]^bin[2];
    assign gray[1] = bin[2]^bin[1];
    assign gray[0] = bin[1]^bin[0];
        
endmodule

Behavioural Level:

module binary_to_gray_4bit_behavioural(bin , gray);
    input [3:0]bin;
    output reg [3:0]gray;
            
    always@(bin)
    begin
            
        case(bin)
             4'b0000 : gray = 4'b0000;
             4'b0001 : gray = 4'b0001;
             4'b0010 : gray = 4'b0011;
             4'b0011 : gray = 4'b0010;
             4'b0100 : gray = 4'b0110;
             4'b0101 : gray = 4'b0111;
             4'b0110 : gray = 4'b0101;
             4'b0111 : gray = 4'b0100;
             4'b1000 : gray = 4'b1100;
             4'b1001 : gray = 4'b1101;
             4'b1010 : gray = 4'b1111;
             4'b1011 : gray = 4'b1110;
             4'b1100 : gray = 4'b1010;
             4'b1101 : gray = 4'b1011;
             4'b1110 : gray = 4'b1001;
             4'b1111 : gray = 4'b1000;
        endcase
    end
            
endmodule

module binary_to_gray_4bit_tb;
